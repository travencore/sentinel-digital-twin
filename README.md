# **Sentinel AI: Agentic Supply Chain Orchestration**

An enterprise-grade **Decision Intelligence** system managing a **$41M global portfolio**. This project mitigates geopolitical disruption through an autonomous **LangGraph** orchestration layer that heals data and simulates strategic pivots.

### **The Intelligence Layer**
* **Sentinel Agent (Data Integrity)**: Identifies "PuckTrick" contamination patterns (5% rate) in freight orders. It autonomously remediates **11 critical anomalies**, including NaT errors and temporal shifts.
* **Navigator Agent (Strategic Pivot)**: Analyzes **35 geopolitical disruption signals**. It executes a China-to-Mexico/Vietnam rerouting strategy, realizing **$3.6M in potential savings** with 92% confidence.

### **Architecture & Tech Stack**
* **Orchestration**: Python & LangGraph for agentic state management.
* **Data Modeling**: GaussianCopulaSynthesizer (SDV) for multi-table synthetic data generation.
* **Graph Layer**: Memgraph (Bolt) mapping 270+ complex supply chain relationships.
* **Frontend**: Next.js 16 (Turbopack) & Framer Motion.
