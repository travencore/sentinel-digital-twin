# 🛡️ Sentinel AI: Agentic Supply Chain Orchestration

**Sentinel AI** is an autonomous Decision Intelligence ecosystem designed to protect a **$41M global portfolio** against geopolitical volatility and data corruption. Moving beyond static reporting, this system leverages a multi-agent **LangGraph** architecture to execute real-time P&L recovery strategies.

---

### 🚀 Key Impact Metrics
* **$3.6M Potential Savings**: Identified via autonomous China-to-Mexico/Vietnam nearshoring pivots.
* **100% Data Integrity**: Sentinel Agent remediated **11 critical anomalies** (NaT/Temporal shifts) across 270 orders.
* **92% Decision Confidence**: Real-time risk modeling across **35 active disruption signals**.

### 🧠 The Multi-Agent Intelligence Layer

#### 1. Sentinel Agent (Self-Healing Data)
* **Function**: Operates as a "Data Integrity Guard," identifying **"PuckTrick"** contamination patterns (5% rate).
* **Outcome**: Cleansed **30 contaminated rows** to ensure the **$41M valuation** remained audit-ready.
